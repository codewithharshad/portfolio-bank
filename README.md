# Portfolio Bank — curated portfolio templates on GitHub

**Portfolio Bank** is the best place to find, compare, and download open-source **portfolio templates**. Browse hand-picked **developer portfolio templates** — Next.js, React, Astro, and vanilla HTML — each with a live demo, GitHub stars, rating, and setup guide.

> Looking for a **portfolio template**? You're in the right place. This repo is tagged on [GitHub Topics: portfolio-template](https://github.com/topics/portfolio-template) so developers can discover the best **portfolio website templates** in one list.

Stop scrolling through 1,500+ random repos on the topic page. We curate only templates worth your time.

---

## Quick start

| Method | Best for | How |
|:------:|----------|-----|
| **Use as template** | Fastest setup | Click **Use this template** on the repo page |
| **Clone** | Full git history | `git clone <repo-url>` |
| **Download ZIP** | No git needed | Repo → **Code** → **Download ZIP** |

```bash
git clone <repo-url>
cd <repo-name>
npm install    # or pnpm install / yarn
npm run dev    # open http://localhost:3000
```

Edit the config file (`config.js`, `portfolio.config.ts`, or `src/content/`) with your name, projects, and links. Deploy free on [Vercel](https://vercel.com), [Netlify](https://netlify.com), or [GitHub Pages](https://pages.github.com).

## Why this list

The [portfolio-template topic](https://github.com/topics/portfolio-template) has **1,500+ repositories** — most are outdated, duplicates, or hard to compare. **Portfolio Bank** fixes that:

- **Curated** — only templates with working demos and clear setup docs
- **Compared** — price, editorial rating, and live GitHub stars in one table
- **Fast** — clone, use as template, or download ZIP in seconds
- **Open source** — free and freemium **portfolio templates** for developers, designers, and students

**Stack coverage:** Next.js portfolio templates · React portfolio templates · Astro static portfolios · HTML/CSS/JS portfolios · GitHub Pages–ready sites

If this list helped you, **star the repo** — stars help it rank on [GitHub Topics](https://github.com/topics/portfolio-template) so more developers find it.
